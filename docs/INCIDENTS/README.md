# Incidents

This directory stores incident reports for SentinelLayer (outages, false-positive spikes, gating regressions, telemetry failures, etc.).

Template:
- Start from `docs/templates/INCIDENT_TEMPLATE.md`.

Naming:
- `docs/INCIDENTS/2026-02-08-short-title.md`

Minimum expectations:
- Timeline with UTC timestamps
- Impact and user-facing symptoms
- Root cause and contributing factors
- Concrete action items with owners
